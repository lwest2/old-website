BlastWay

This was a game I created with the Phaser plugin during my first year of University, I plan to improve this in the future. The object of this game is to try and get as many laps as possible in the given time. Avoid the bombs, especially the red bombs that spawn in on lap 2. The explosions also need to be avoided at all costs! Collect the coins and make it back to the finishing line to complete a lap.

Buttons

  • Mouse to steer the car in a direction.
  • R key to restart.
  • P key to pause.
  • M key to mute.
